### Módulo 2: Instalación y Configuración

#### 2.1. Instalación de Git

Antes de comenzar a trabajar con Git, necesitas instalarlo en tu sistema. Git es compatible con una variedad de sistemas operativos, incluyendo Windows, macOS y Linux.

#### Instalación de Git en Windows

1. **Descargar el instalador de Git**:
   - Ve al sitio web oficial de Git para Windows: [Git for Windows](https://gitforwindows.org/).
   - Haz clic en el botón de descarga para obtener el instalador.

   ![Descarga de Git para Windows](/img/img1.png)

2. **Ejecutar el instalador**:
   - Una vez descargado, navega hasta la ubicación donde se guardó el archivo descargado (por lo general, la carpeta "Descargas").
   - Haz doble clic en el archivo .exe descargado para iniciar el proceso de instalación.

   ![Instalador de Git para Windows](/img/img2.png)

3. **Configuración de opciones**:
   - Durante la instalación, se te presentarán diversas opciones. Por lo general, las opciones predeterminadas son adecuadas para la mayoría de los usuarios. Puedes seguir adelante haciendo clic en "Next" en cada paso.
   - Si deseas personalizar alguna opción, como la ubicación de instalación o las opciones del menú contextual, puedes hacerlo según tus preferencias.

4. **Finalizar la instalación**:
   - Una vez completada la instalación, haz clic en "Finish" para cerrar el instalador.

   ![Finalización de la instalación de Git para Windows](/img/img3.png)

5. **Verificar la instalación**:
   - Abre el símbolo del sistema (cmd) o PowerShell.
   - Escribe el siguiente comando y presiona Enter: `git --version`.
   - Deberías ver la versión de Git instalada si todo se ha configurado correctamente.

   ![Verificación de la instalación de Git en Windows](/img/img4.png)


#### Instalación de Git en macOS

1. **Instalación con Homebrew** (recomendado):
   - Abre la Terminal.
   - Ejecuta el siguiente comando para instalar Homebrew si aún no lo tienes: `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`.
   - Después de instalar Homebrew, ejecuta el siguiente comando para instalar Git: `brew install git`.

2. **Instalación desde el paquete de instalación de Git**:
   - Ve al sitio web oficial de Git para macOS: [Git para macOS](https://git-scm.com/download/mac).
   - Descarga el paquete de instalación para macOS.
   - Haz doble clic en el paquete descargado y sigue las instrucciones en pantalla para completar la instalación.

3. **Verificar la instalación**:
   - Abre la Terminal.
   - Escribe `git --version` y presiona Enter. Deberías ver la versión de Git instalada si todo se ha configurado correctamente.


#### Instalación de Git en Linux (Ubuntu/Debian)

1. **Instalación a través de apt**:
   - Abre la Terminal.
   - Ejecuta el siguiente comando para actualizar el índice de paquetes: `sudo apt update`.
   - Luego, instala Git escribiendo: `sudo apt install git`.

2. **Verificar la instalación**:
   - Después de la instalación, escribe `git --version` en la Terminal y presiona Enter. Deberías ver la versión de Git instalada si todo se ha configurado correctamente.


#### 2.2. Configuración inicial de Git

Después de instalar Git, es importante configurarlo correctamente antes de comenzar a trabajar con repositorios.

- **Configuración global**:
  - Especifica tu nombre de usuario y dirección de correo electrónico que se asociarán con tus commits en todos los repositorios. Esto se puede hacer ejecutando los siguientes comandos:
    ```
    git config --global user.name "Tu Nombre"
    git config --global user.email "tu@email.com"
    ```
  - También puedes configurar otros aspectos globales, como el editor de texto predeterminado (`git config --global core.editor`) y la configuración de color (`git config --global color.ui`).

- **Configuración local**:
  - Si deseas configurar opciones específicas para un repositorio en particular, puedes omitir la opción `--global` al ejecutar los comandos de configuración.

- **Verificación de configuración**:
  - Para verificar la configuración actual de Git, puedes ejecutar `git config --list`.

- **Opciones adicionales de configuración**:
  - Git ofrece una variedad de opciones de configuración que puedes explorar para personalizar tu entorno de trabajo, como alias de comandos (`git config --global alias.<alias> <comando>`), ignorar archivos específicos (`git config --global core.excludesfile`), etc.

En este módulo, has aprendido cómo instalar Git en tu sistema operativo y cómo configurar correctamente tu identidad y otras opciones de configuración. Configurar Git de manera adecuada es esencial para un flujo de trabajo eficiente y para garantizar que tus contribuciones sean adecuadamente identificadas en los repositorios.
